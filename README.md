# Veteran Lung Cancer Survival Analysis

A survival analysis of the U.S. Veterans Administration lung cancer trial
(Kalbfleisch & Prentice, 1980), comparing standard and experimental
chemotherapy in 137 male patients with advanced, inoperable lung cancer. It
uses Kaplan-Meier estimation and Cox proportional hazards models to work out
what predicts survival.

A portfolio piece built in R and Quarto.

**Live site:** <https://julian-chung.github.io/veteran-lung-cancer-coxph/>
