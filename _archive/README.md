# Archive

`veteran-lung-cancer-coxph.qmd` is the original teaching-and-learning guide: a
walkthrough of the same Cox proportional hazards analysis with collapsible
explanations of the theory, code, and diagnostics.

It has been unlinked from the portfolio site. It is not in the navbar, and the
leading underscore in `_archive/` tells Quarto to skip the directory entirely,
so nothing here is rendered or published. It is kept for reuse in a separate
teaching repo.
